# Ícone iOS 180x180px

Para corrigir o problema do ícone não carregar no iOS, é necessário criar um ícone de 180x180 pixels.

## Como criar o ícone

1. Use o arquivo `MvCash_Icone.png` como base (localizado em `apps/frontend/public/`)
2. Redimensione a imagem para exatamente 180x180 pixels
3. Salve como `icon-180x180.png` na pasta `apps/frontend/public/icons/`

## Ferramentas recomendadas

- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [PWA Builder](https://www.pwabuilder.com/imageGenerator)
- Qualquer editor de imagens (Photoshop, GIMP, etc.)

## Nota

O iOS requer especificamente um ícone de 180x180px para funcionar corretamente quando o app é adicionado à tela inicial. Sem este ícone, o iOS pode não exibir o ícone corretamente.
